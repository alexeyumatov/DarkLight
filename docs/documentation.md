# DarkLight — Дизайн-документ

---

## 1. Концепция

- **Жанр:** 2D платформер с элементами roguelite (прокачка между уровнями)
- **Сеттинг:** Мрачная башня. Нижний этаж — таверна, верхние этажи — комнаты с врагами
- **Цель игрока:** Добраться до вершины башни, проходя уровни-комнаты и накапливая монеты на прокачку навыков

---

## 2. Структура игры

- **Глобальный экран** — башня снизу вверх, каждый этаж является одним уровнем-комнатой
- **Таверна (нижний этаж)** — меню прокачки навыков, доступна между уровнями
- **Уровень (комната)** — пол, стены, лестницы, враги, монеты, целебный фонтан, шипованная плита, портал

---

## 3. Игрок

| Параметр     | Значение                                   |
| ------------ | ------------------------------------------ |
| Здоровье     | 100 HP                                     |
| Передвижение | Бег влево/вправо, прыжок                   |
| Атака        | Стрельба пулями                            |
| Мана         | Накапливается при попадании пули в монстра |

### Суперспособность

При полной мане игрок активирует суперспособность, которая мгновенно уничтожает всех монстров, находящихся перед ним по горизонтали в пределах комнаты.

### Навыки (прокачиваются в таверне за монеты)

- **Дополнительная броня** — снижает входящий урон от атак противников
- **Дополнительный урон пуль** — увеличивает урон каждой выпущенной пули
- **Несколько пуль за выстрел** — игрок выпускает 2–3 пули одновременно

---

## 4. Объекты уровня

### 4.1. Монстры

| Тип     | HP      | Атака                            | Особенность                     |
| ------- | ------- | -------------------------------- | ------------------------------- |
| Слабый  | Низкое  | Урон при контакте (малый)        | —                               |
| Средний | Среднее | Стреляет в игрока (малый урон)   | Держит дистанцию от игрока      |
| Сильный | Высокое | Стреляет в игрока (большой урон) | Накладывает яд (урон в секунду) |

**Яд (сильный монстр):** снимает HP игрока каждую секунду в течение нескольких секунд после попадания.

### 4.2. Монеты

- Появляются на уровне только при первом прохождении
- При повторном прохождении монеты отсутствуют
- Собранные монеты тратятся в таверне на прокачку навыков

### 4.3. Целебный фонтан

- При контакте с игроком восстанавливает часть его HP
- Одноразовый — после использования деактивируется и не работает повторно

### 4.4. Шипованная плита

- При наступании игрока активирует ловушку
- По всей ширине комнаты пролетают шипы, наносящие урон игроку при попадании

### 4.5. Портал

- Конечная точка уровня
- При входе происходит переход на следующий уровень или возврат в таверну

### 4.6. Лестницы

- Игрок может карабкаться по ним вверх и вниз
- Соединяют платформы внутри комнаты

## 5. Игровые сцены (MonoGame)

| Сцена           | Содержимое                               |
| --------------- | ---------------------------------------- |
| `MainMenuScene` | Заставка, кнопка старта игры             |
| `TavernScene`   | Интерфейс прокачки навыков за монеты     |
| `GameScene`     | Текущий активный уровень-комната         |
| `GameOverScene` | Экран гибели игрока с опцией перезапуска |

---

## 6. Архитектура проекта

| Папка / Файл | Назначение                                           |
| ------------ | ---------------------------------------------------- |
| `Content/`   | Спрайты, тайлы, шрифты (Content Pipeline)            |
| `Scenes/`    | MainMenuScene, TavernScene, GameScene, GameOverScene |
| `Entities/`  | Player, Enemy (Weak / Medium / Strong), Coin, Bullet |
| `Objects/`   | Fountain, SpikePlate, Portal, Ladder                 |
| `Managers/`  | SceneManager, CollisionManager, CameraManager        |
| `UI/`        | HealthBar, ManaBar, CoinCounter                      |
| `Game1.cs`   | Точка входа, основные методы Update / Draw           |

## 7. Список спрайтов и ассетов

### Игрок

- Idle, Walk, Death

### Средний монстр

- Idle, Walk, Shoot, Death

### Сильный монстр

- Idle, Walk, Shoot, Death

- Тайлы: пол, стена, фон комнаты
